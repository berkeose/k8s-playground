# k8s-from-scratch

A hands-on Kubernetes learning repository where I document my notes, lab exercises, YAML examples, and mini projects while learning Kubernetes from scratch.

## Purpose

This repository is my personal Kubernetes learning journey.  
The goal is not only to watch a course, but also to:

- understand Kubernetes concepts clearly
- practice them in a local lab environment
- write structured notes
- create reusable YAML examples
- build a portfolio-style GitHub project

## Learning Approach

I am learning Kubernetes step by step through:

- course-based study
- hands-on lab practice
- writing my own notes
- testing YAML manifests
- documenting what I learned after each topic

## Repository Structure

- `00-roadmap/` → learning plan and roadmap
- `01-fundamentals/` → core Kubernetes concepts
- `02-labs/` → hands-on exercises and notes
- `03-yaml-examples/` → Kubernetes manifest examples
- `04-cheatsheets/` → command references and quick notes
- `assets/` → diagrams and visuals
